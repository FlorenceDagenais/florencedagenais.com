Reflections

Dans le cadre de ce projet, j’ai eu la chance de travailler avec une équipe de 13 personnes, 9 artistes multidisciplinaires et 4 programmeurs.

Défis techniques:

  • Première utilisation de Unreal Engine 4
  • Jeu Android avec limitations mobile
  • Jeu multiplayer

Mes tâches sur le projet consistaient à:

  • Me familiariser avec les nouvelles technologies et leur application dans le cadre de notre projet de jeu.
  • Level Art
  • Être le lien entre les programmeurs et les artistes.
  • Unreal 4 UMG tool
  • Utilisation de Cascade pour les FX
  • Créer les master materials pour un système de Material Instance
  • Faire les state machines de base des personnages (idle, marcher, courir, sauter)

Dans le cadre du projet j’ai également participé au:

  • Game Design
  • Level Design
