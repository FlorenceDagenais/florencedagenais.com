Hereafter

Cliquez ici pour voir le trailer du jeu

Défis techniques:

  • élaboration d'un controller permettant de controller les différents personnages ainsi que de les controller en alternance
  • Programmation par Blueprint
  • Création des Master Shaders permetant de faire la transition entre les visions
  • Programmation de l'interface utilisateur ainsi qu'un system d'inventaire

Mes tâches sur le projet consistaient à:

  • Élaborer et concevoir un state machine permettant de viser avec la lanterne
  • Création d'un state machines de base pour les différents personnages
  • Créer les différents master materials pour un système de Material Instance
  • Création d'un system de material layers afin d'ajouter différents matériaux de saleté de façon procédurale
  • Créer un system pour les lumières, les décals et le post process afin de blender entre plusieurs states
  • Intégration des différents éléments audio
  • Animation d'éléments interractifs
  • Création de system de particules
  • Programation du system pour les divres éléments interactif
